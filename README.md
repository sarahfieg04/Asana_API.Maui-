Asana3_API.Maui - MAUI Task and Project Management Application 
Asana3 is a .NET MAUI application which allows users to manage projects and their ToDo task. 
This application communicates with a local API using HTTP request and then displays data through an MVVM architecture 

User capabilites include: 
- Creating and managing ToDo items
- Creating and managing Project items
- Assigning ToDos to Projects
- Updating, Deleting, and Viewing tasks
- Assigning Due Dates to tasks

# Project Structure: 
- Asana3: Console / API runner
- Asana3.Library: Models + Services
- Asana3.Maui: MAUI UI Application

# How to run: 
1. Download the files 
2. Open the solution "Asana3.sln"
3. Restore Packages if needed
4. Set startup project to "Asana3.Maui
5. Then build and run

- The MAUI UI should then launch and display
- ToDo tasks
- Project list
- Project task assignments
